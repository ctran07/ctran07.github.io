Assignment 7: Computer control

Circuit operation

This circuit reads the x- and y- values from a joystick and draws an ellipse mapped to p5. Through keyboard input, the brightness of an LED can be controlled.

A0 reads the x- values, and A1 reads the y- values from the joystick. An LED is connected to pin 5, which reads from keyboard input.

Schematic


For blue LEDs:
3.3V drop, 5V power source, 20 mA
V = 5-3.3 = 1.7 V
V = I x R = 1.7 = 0.02 x R
R = 1.7/0.02 = 85 Ω

I chose to use a 1K Ω resistor to limit the current flowing to the LED, much more than the 85 Ω requirement.
